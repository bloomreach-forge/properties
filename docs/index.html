Introduction

Properties plugin is deprecated for usage on behalf of site labels

Instead of using this plugin for site label localization, we recommend the usage of resource bundles, see HST Dynamic Resource Bundles support and the HST Resource Bundle Editor Plugin

Usage of the plugin on behalf 'system' properties can still be applicable for your project.

What is the properties component?

The properties component allows you to create simple CMS documents having a flat list of name/value pairs. It also provides easy locating and using these documents in your HST site frontend.

Code base on GitHub

The code of the project is on GitHub at https://github.com/onehippo-forge/properties/.
